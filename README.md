This project measures walking access from every MBTA station to nearby parks using network-based pedestrian isochrones.

## Key finding

98.6% of MBTA stations reach a park within a 15-minute walk.

## At a glance

- 282 stations analyzed
- 5, 10, and 15 minute walking isochrones
- Only 2 stations have no park access
